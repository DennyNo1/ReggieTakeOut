# 工程简介
简化版的一个外卖平台，作为我找工作的一个练手项目，最近为了面试又拿了起来。绝大部分来源于黑马（一家网络培训机构）。
整个系统分为前端和后端。前端就是消费者端，仅限于在一个商家购买。后端就是商家端。
